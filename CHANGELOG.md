# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-07-06

### Added

- Initial release of the FastAPI project generator.
- Includes FastAPI, Uvicorn, Pydantic, and `datamodel-code-generator`.
- Basic project structure with `app` and `schemas` directories.
- `Makefile` for dependency installation and model generation.
- `.gitignore` file.
- `README.md` for the generated project.
